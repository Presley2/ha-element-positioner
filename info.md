## Element Positioner

Visueller Drag & Drop Editor für `picture-elements` Dashboards.

Elemente auf dem Grundriss verschieben, skalieren und bearbeiten — direkt im Browser, ohne Code.

**Funktionen:** Drag & Drop · Pfeiltasten · Mehrfachauswahl · YAML/JSON Editor · Undo/Redo · Snap-Grid · Resize · Format-Painter · Element kopieren zwischen Tabs

> Nach der Installation: `drag_editor_panel.js` manuell in `/config/www/` kopieren und `panel_custom` in `configuration.yaml` eintragen. Siehe README für Details.
